# EcoReward - Smart Waste Management System
A premium web-based reward system built with **Flask**, **HTML5**, and **Modern CSS**.

## Features
- **Python Backend**: Lightweight and secure Flask server.
- **Modern UI**: Desktop-friendly design with glassmorphism and eco-friendly aesthetics.
- **Firebase Integration**: Ready for Firestore and Auth via `firebase-admin`.
- **Responsive**: Fully responsive design that works on desktops, tablets, and mobiles.

## Prerequisites
- Python 3.8+
- Firebase Project

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Set up Firebase:
   - Place your `serviceAccountKey.json` in the project root.
   - Follow instructions in `Firebase_Python_Setup.md`.
3. Run the application:
   ```bash
   python app.py
   ```
4. Open your browser to `http://127.0.0.1:5000`.

## Folder Structure
- `app.py`: Backend logic and routing.
- `templates/`: HTML Jinja2 templates.
- `static/css/`: Premium Vanilla CSS styling.
- `requirements.txt`: Python dependencies.
